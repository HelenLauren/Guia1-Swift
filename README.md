# Guia1-Swift
Atividade prática para a matéria de Desenvolvimento Mobile IOS

Durante a realização dos exercícios e do desafio em Swift, foram praticados conceitos fundamentais da linguagem, como declaração de variáveis, tipos de dados, operações básicas e uso de estruturas de controle como if, switch e loops. Também foram utilizados arrays e dicionários para armazenar e manipular dados, além de tuplas para organizar informações de forma simples. No desafio final, esses conceitos foram aplicados juntos com structs, funções e optionals, permitindo criar um pequeno sistema de gerenciamento de livros que cadastra, lista, altera disponibilidade e busca registros de forma segura.
